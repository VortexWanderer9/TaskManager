## Task manager

A simple task manager application built with React and TypeScript. It allows users to create, edit, and delete tasks, as well as mark them as completed.

## Plan for the project

1. create UI/UX on Figma
2. first create the basic structure of the application using React
3. implement state management using React's useState and useEffect hooks
4. add functionality to create, edit, delete, and mark tasks as completed
